## YAGNI

Don't add features, customisation points, or extensions unless they're
explicitly asked for. This could should avoid premature abstractions or
generalisation. Some duplication of code is okay if the intent differs. Don't
fight with the declarative nature of Nix to avoid duplicating a simple handful
of lines of code. Refactors can always be done later when the code matures.
